# Shopping Website Made From Django

- A shopping website which has been made form the Django framework of Python along with HTML, CSS, JS, BOOTSTRAP.
- A specific item can be searched, and added to the cart. The items which are added to the cart are stored in the browsers local storage.
- PayTM Payment integration feature has also been added (for more information, see the PayTM developer docs and the PayTM integration API).

## Features Included in this shopping website project

- Buy products from the main page of the website
- Track your order 
- Read blogs
- Contact Us Page
- PayTM gateway integration included during checkout after selecting the prodcuts

## Tech Stacks Used

- Django
- Javascript
- Jquery
- Python
- HTML
- CSS

## Django Specific Commands

- For the installation of Django, run the following command in the terminal:
```
pip install django
```
- After installation, check whether Django is recognised or not by running the following command in the terminal:
```
django --version
```
- For running the project on localhost:8000, run the following command in the terminal:
```
python manage.py runserver
```
- For seeing the other available commands, type
```
python manage.py help
```
